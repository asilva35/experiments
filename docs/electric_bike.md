Act as a Senior Creative Developer. Create a **'E-Bike Range & Route Planner'** using React and Tailwind CSS.

**The Concept:**
A sleek tool for an electric bicycle company where users can simulate a trip and see their estimated battery level at arrival.

**Features to build:**
1.  **Map Integration:** A map area where a route is displayed (use a visual placeholder for the route).
2.  **Bike Selector:** A toggle or cards to select between 'Model S (1 seat)', 'Model M (2 seats)', and 'Cargo L (3 seats)'.
3.  **Load Slider:** A way to select how many people are on the bike.
4.  **Elevation Toggle:** A switch to simulate 'Flat Terrain' vs 'Hilly Terrain' (this will affect the calculation).
5.  **Dynamic Result:** A big, beautiful circular progress bar or battery icon that updates in real-time showing: 'Remaining Battery: 65%'.

**Design:**
* **Palette:** Electric Yellow (#facc15) and Dark Charcoal (#18181b).
* **Vibe:** High-tech, adventurous, and modern.
* **Summary:** Show a summary of the trip: Distance, Estimated Time, and Elevation Gain.

Include a **'Buy this Bike'** call to action that changes based on the selected model.

Create the code in `src/pages/electric_bike/ElectricBike.tsx` file.
